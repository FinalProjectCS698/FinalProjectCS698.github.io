Project Summary

This research addresses the need for more efficient insect population surveying methods, aiming to overcome the limitations of traditional labor-intensive approaches. Utilizing remote sensing, particularly focusing on wingbeat frequency (WBF), offers a transformative solution, enhancing precision in insect identification by exploring discriminative factors like body-wing ratio. The end goal is to create a system, involving a Raspberry Pi and infrared sensor, to collect insect transit data remotely. This project serves as preliminary research into how we might accomplish this. The dataset comprises six CSV files, processed using Python scripts, and the final phase involves building machine learning models to predict insect families and sex based on features, evaluating them using various metrics.
